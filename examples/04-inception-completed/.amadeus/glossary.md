# 用語集

## 用語

| 用語 | 説明 | 状態 |
|---|---|---|
| ECサイト | 商品を公開し、購入者が商品を選択して注文できる販売チャネル。 | 確認済み |
| 商品管理 | ECサイトで扱う商品情報を管理する領域。 | 確認済み |
| 顧客管理 | 購入者または顧客に関する情報を管理する領域。 | 確認済み |
| 入荷管理 | 商品が販売可能になる前の入荷を管理する領域。 | 確認済み |
| 販売管理 | 商品選択から注文作成までの販売活動を管理する領域。 | 確認済み |
| 出荷管理 | 注文後の商品発送を管理する領域。 | 確認済み |
| 販売可能在庫 | 購入フローで販売可否を確認する在庫。具体的な在庫引当は未確認である。 | 確認済み |
