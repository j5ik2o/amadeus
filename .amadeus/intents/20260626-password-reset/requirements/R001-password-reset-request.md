# R001：利用者がパスワード再設定を要求できる

## 要求

利用者は、自分のアカウントに対してパスワード再設定を要求できる。

## 受け入れ条件

- アカウントに紐づく連絡先が存在する場合、パスワード再設定を要求できる。
- パスワード再設定要求により再設定トークンが発行される。
- 再設定トークンは期限付きである。
- 存在しないアカウントに対する要求では、アカウントの存在有無を推測できる応答を返さない。
