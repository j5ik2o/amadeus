# R003: 適用成果物範囲

## 要求

- 参照リンク化方針を適用する成果物範囲を説明できる。
- template、example、traceability、decisions、Functional Design、既存成果物のうち、今回の Intent で扱う範囲と対象外を分けられる。
- Functional Design の内容変更、Unit と Bolt の再分割、Domain Map と Context Map の採用判断変更、machine-readable evidence の導入を対象外として維持する。

## 受け入れ条件

- source skill と昇格先成果物のどちらを更新対象にするかを読める。
- example snapshot と既存 `.amadeus/` 成果物の扱いを読める。
- 内容変更ではなく参照リンク化方針の適用として扱う境界を読める。
