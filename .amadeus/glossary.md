# 用語集

## 利用者

利用者。システムにログインし、サービスを利用する主体。

## アカウント

利用者に紐づく認証・契約・設定の管理単位。

## 認証情報

ログイン認証に使う秘密情報。パスワード、外部認証プロバイダー連携情報などを含む。

## 再設定トークン

パスワード再設定に使う期限付き・単回利用のトークン。

## パスワード再設定

アカウントの認証情報を再設定する手続き。
再設定トークンの発行、検証、認証情報更新を含む。

## 避ける語

### 顧客

このプロダクトでは曖昧なので原則使わない。
文脈に応じて「利用者」「アカウント所有者」「請求連絡先」を使い分ける。

## 禁止ワード

### runtime / Runtime

原則禁止。
スキルが実際に使われる時点、またはスクリプトが実行される時点は「実行時」と書く。

例外は、既存コード、固有名、外部仕様名として必要な場合に限る。

### runtime dependency

原則禁止。
スキルを実行するために必要な依存は「実行時依存」と書く。

開発、評価、昇格判断だけに必要な依存とは分ける。

### 正本

原則禁止。
判断や作業の基準を表す場合は「参照元」「基準」「定義元」「管理元」のように、文脈に合う語へ置き換える。

例外は、引用、法律文書、契約文書など、その語を変えると意味が変わる場合に限る。
