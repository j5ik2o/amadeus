# Lints

このディレクトリは、Amadeus 固有の lint を置く場所である。

`lints/check.ts` は `lints/*/check.ts` を検出して実行する。
各 lint は、原則として `--check` と `--report` を提供する。
閾値のように判定基準そのものを表すオプションは、lint ごとに持ってよい。

各 lint は、既存違反を記録して以後の判定から除外する仕組みを持たない。
違反が出た場合は、対象コードを修正する。
判定基準が不適切な場合は、例外を追加せず、lint 仕様を見直す。

## public-type-file

1 ファイルに複数の公開型を置かないことを検査する。
公開型を支える非公開実装は、同じファイルに置いてよい。

## ts-complexity

TypeScript 関数の循環的複雑度を検査する。
閾値を超えた場合は、条件分岐の整理や関数の分割によって解消する。
